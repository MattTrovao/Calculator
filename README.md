# Calculator

Projeto para calcular:
- Valor da Hora trabalhada
- Taxa de Juros
- IMC

# Rodar o Projeto

- Faça download do projeto
- Navegue até a pasta do projeto `cd Calculator`
- Rode o comando `npm i`
- Após instalar depências rode o comando `npm run dev`
